# linusthe.dev

This site is build with [Astro](https://docs.astro.build/en/getting-started/).

It is still under development! 👷‍♂️

<br>

## How to run

I use [pnpm](https://pnpm.io/) as a package manager.

```bash
# install dependencies
pnpm install

# run locally
pnpm dev
```
